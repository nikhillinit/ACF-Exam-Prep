═══════════════════════════════════════════════════════════
KEY FILE PURPOSES
═══════════════════════════════════════════════════════════

CONFIGURATION:
• package.json          → Dependencies, scripts, metadata
• webpack.config.js     → Bundle configuration
• babel.config.js       → JSX/ES6+ transpilation
• .env.example          → Environment template

CORE DATA:
• archetype-signals.json → Archetype definitions, keywords, 
                            time allocations, playbook mappings
• keyword-mappings.json  → Keyword-to-archetype lookup table
• problems-index.json    → Mock exam catalog with metadata
• tier-definitions.json  → Tier 1/2 rules, mastery levels

CORE LOGIC:
• archetypeScanner.js    → Scans text, identifies archetypes
• keywordMatcher.js      → Matches keywords with confidence
• workflowEngine.js      → Generates 5-step workflow

MAIN COMPONENTS:
• ReconView.jsx          → Main reconnaissance interface
• Scanner.jsx            → Displays scan results
• HybridSequencer.jsx    → Shows hybrid solving sequence
• ArchetypeMapper.jsx    → Maps archetypes to resources

STUDY MATERIALS:
• Corporate_Finance_Templates.xlsx → 11 Excel worksheets
• Mock_Solutions.md      → Comprehensive solution keys
• Fall_2024_Final.pdf    → Latest exam
• pdfs/                  → Archive of past exams

═══════════════════════════════════════════════════════════
DATA FLOW
═══════════════════════════════════════════════════════════

1. USER PASTES PROBLEM TEXT
   ↓
2. archetypeScanner.js
   ├→ Extracts keywords using keywordMatcher.js
   ├→ Scores archetypes using archetype-signals.json
   ├→ Detects hybrid patterns
   └→ Returns scan results
   ↓
3. ReconView.jsx DISPLAYS:
   ├→ Scanner.jsx (scan results)
   ├→ HybridSequencer.jsx (if hybrid)
   ├→ ArchetypeMapper.jsx (resource mapping)
   └→ TimeAllocator.jsx (time budget)
   ↓
4. USER CLICKS "Open Excel"
   ↓
5. ExcelLauncher.jsx
   └→ Opens Corporate_Finance_Templates.xlsx at correct tab
   ↓
6. USER SOLVES PROBLEM
   ↓
7. SolutionCheck.jsx VERIFIES
   └→ Uses pointCalculator.js for scoring
