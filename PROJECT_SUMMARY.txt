═══════════════════════════════════════════════════════════════
CORE ALGORITHM: archetypeScanner.js
═══════════════════════════════════════════════════════════════

Input: Problem text string
Process:
  1. Extract keywords
  2. Score each archetype (matches * confidence)
  3. Detect hybrid patterns
  4. Rank archetypes by score
  5. Determine solving sequence
Output: {archetypes[], keywords[], isHybrid, tier, points}

Complexity: O(n*m) where n=text length, m=keywords
Average runtime: <100ms
